## Chapter5 循环和关系表达式



### 基于范围的 for 循环

```
//基于范围的 for 循环,但无法修改数组或容器内的元素
double prices[5] = {4.1, 4.2, 4.3, 4.4, 4.5}
for (double c : prices) {
	cout << x << std::endl;
}

//修改元素
for (double &x : prices) {
	x *= 0.8;
}
```
